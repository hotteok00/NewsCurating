# 주간 보고서 피드백 (2026-03-09 ~ 2026-03-15)

## 개선 요청 사항

### 1. 유튜브 영상 처리
- 유튜브 영상은 제목만으로 내용을 추측하지 말 것
- 자막/요약(summary)이 충분하지 않으면 "영상 확인 필요"로 표시
- 요약이 광고/홍보 문구 위주인 경우 해당 내용은 기사 요약에서 제외

### 2. 속보 정리 방식
- 속보 기사가 너무 많이 나열되어 가독성이 떨어짐
- 같은 사건의 속보는 타임라인 형태로 묶어서 시간순 정리
- 예: "미국-이란 전쟁" 관련 속보 → 날짜별 타임라인으로 통합

### 3. 대주제 통합
- 같은 주제(예: 미국-이란 전쟁)가 정치, 비즈니스, 기술 등 여러 카테고리에 분산됨
- 해당 주의 핵심 대주제는 별도 섹션으로 묶어서 다각도 분석 제공
- 카테고리 분류보다 주제 중심 정리를 우선시할 것

### 4. 핵심 포인트 구체성
- "중요하다", "주목할 만하다" 같은 일반적 표현 지양
- 구체적 수치, 팩트, 데이터를 반드시 포함
- 예: "유가 상승" → "유가 배럴당 $90 돌파, 주간 15% 상승"
- 예: "대규모 해고" → "메타 약 16,000명(전체 20%) 감원 검토"
